# Conference Event Planner

This is a practice project from the ["Developing Front-End Apps with React"](https://www.coursera.org/learn/developing-frontend-apps-with-react) course by IBM on Coursera.

## Scenario

Alejandre manages a venue for business conferences. Her parent company, "BudgetEase," wants to hire you to develop a website so BudgetEase customers can price their conference events easily.

The application's requirements include allowing users to select and price the rooms in the conference center, add-on selections like microphones and projectors, and meals for a given number of guests.

## Features

- Select and price conference rooms (e.g., Conference Room, Auditorium Hall, Presentation Room)
- Add audio-visual equipment (e.g., projectors, speakers, microphones, whiteboards, signage)
- Choose meals for guests
- Calculate total cost dynamically
- User-friendly interface for event planning

## Technologies Used

- React
- Redux Toolkit for state management
- Vite for build tooling
- JavaScript (ES6+)
- CSS for styling

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/conference-event-planner.git
   cd conference-event-planner
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## Usage

- Select the desired venue rooms and quantities.
- Add audio-visual equipment as needed.
- Choose meals for the number of guests.
- View the total cost updated in real-time.
